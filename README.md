# Власний застосунок-кодек для компресiї даних
Цей репозиторій містить реалізацію компʼютерного проєтку з дискретної математики.

## Мета роботи
Реалiзувати найпоширенiшi алгоритми компресiї, дослiдити їхню ефективнiсть та створити власний застосунок-кодек,
подiбний до популярних форматiв збереження медiаданих

## Завдання
+ Ознайомитися з основними алгоритмами компресiї даних.
+ Реалiзувати з нуля класичнi методи на основi теорiї кодування.
+ Створити застосунок-кодек для стиснення та розпакування рiзних типiв даних.
+ Провести тестування на реальних даних i зробити висновки щодо ефективностi.
